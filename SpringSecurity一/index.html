  • йқһеёёз®ҖеҚ•зҡ„еҠ еҜҶ APIпјӣ

  • дёҚи·ҹд»»дҪ•зҡ„жЎҶжһ¶жҲ–иҖ…е®№еҷЁжҚҶз»‘пјҢеҸҜд»ҘзӢ¬з«ӢиҝҗиЎҢгҖӮ

/**
 * йӘҢиҜҒз ҒйӘҢиҜҒfilter йңҖиҰҒз»§жүҝOncePerRequestFilterзЎ®дҝқеңЁдёҖж¬ЎиҜ·жұӮеҸӘйҖҡиҝҮдёҖж¬ЎfilterпјҢиҖҢдёҚ
 * йңҖиҰҒйҮҚеӨҚжү§иЎҢ
 *
 * @author pengwangwang
 * @date 2022/10/11 09:39
 **/
@Component
public class ValidateCodeFilter extends OncePerRequestFilter {

    @Autowired
    StringRedisTemplate stringRedisTemplate;


    @Override
    protected void doFilterInternal(HttpServletRequest httpServletRequest, HttpServletResponse httpServletResponse, FilterChain filterChain) throws ServletException, IOException {
        // еҲӨж–ӯжҳҜеҗҰжҳҜзҷ»еҪ•иҜ·жұӮ
        if ("/login".equals(httpServletRequest.getRequestURI()) && "POST".equals(httpServletRequest.getMethod())) {
            try {
                validate(httpServletRequest);
            } catch (ValidateCodeException e) {
                e.printStackTrace();
                return;
            }
        }
        //еҰӮжһңдёҚжҳҜзҷ»еҪ•иҜ·жұӮпјҢзӣҙжҺҘи°ғз”ЁеҗҺйқўзҡ„иҝҮж»ӨеҷЁй“ҫ
        filterChain.doFilter(httpServletRequest, httpServletResponse);
    }


    public void validate(HttpServletRequest request) throws ValidateCodeException {
        // иҺ·еҸ–ip
        String ip = request.getRemoteAddr();
        //жӢјжҺҘredisзҡ„key
        String key = "code_" + ip;
        // иҺ·еҸ–redisдёӯзҡ„йӘҢиҜҒз Ғ
        String code = stringRedisTemplate.opsForValue().get(key);
        String redisImageCode = request.getParameter("imageCode");
        if (StringUtils.hasText(redisImageCode)) {
            throw new ValidateCodeException("йӘҢиҜҒз ҒдёҚиғҪдёәз©ә");
        }
        if (redisImageCode == null) {
            throw new ValidateCodeException("йӘҢиҜҒз Ғе·ІиҝҮжңҹ");
        }
        if (!redisImageCode.equals(code)) {
            throw new ValidateCodeException("йӘҢиҜҒз ҒдёҚжӯЈзЎ®");
        }
        // д»Һredisдёӯз§»йҷӨimageCode
        stringRedisTemplate.delete(key);
    }

}

csrfйҳІжҠӨжңәеҲ¶

CRSFжҳҜд»Җд№Ҳ?

CSRFпјҲCross-site request forgeryпјүпјҢдёӯж–ҮеҗҚз§°пјҡи·Ёз«ҷиҜ·жұӮдјӘйҖ  дҪ иҝҷеҸҜд»Ҙиҝҷд№ҲзҗҶи§ЈCSRFж”»еҮ»пјҡж”»еҮ»иҖ…зӣ—з”ЁдәҶдҪ зҡ„иә«д»ҪпјҢд»ҘдҪ зҡ„еҗҚд№үеҸ‘йҖҒжҒ¶ж„ҸиҜ·жұӮгҖӮCSRFиғҪеӨҹеҒҡзҡ„ дәӢжғ…еҢ…жӢ¬пјҡд»ҘдҪ еҗҚд№үеҸ‘йҖҒйӮ®д»¶пјҢеҸ‘ж¶ҲжҒҜпјҢзӣ—еҸ–дҪ зҡ„иҙҰеҸ·пјҢз”ҡиҮідәҺиҙӯд№°е•Ҷе“ҒпјҢиҷҡжӢҹиҙ§еёҒиҪ¬иҙҰвҖҰвҖҰйҖ жҲҗзҡ„й—® йўҳеҢ…жӢ¬пјҡдёӘдәәйҡҗз§Ғжі„йңІд»ҘеҸҠиҙўдә§е®үе…ЁгҖӮ

CRSFеҺҹзҗҶ

1.зҷ»еҪ•еҸ—дҝЎд»»зҪ‘з«ҷAпјҢе№¶еңЁжң¬ең°з”ҹжҲҗCookieгҖӮ

2.еңЁдёҚзҷ»еҮәAзҡ„жғ…еҶөдёӢпјҢи®ҝй—®еҚұйҷ©зҪ‘з«ҷBгҖӮ

3.и§ҰеҸ‘зҪ‘з«ҷBдёӯзҡ„дёҖдәӣе…ғзҙ

3.еңЁHTTPеӨҙдёӯиҮӘе®ҡд№үеұһжҖ§е№¶йӘҢиҜҒ

    /**
     * и·Ёеҹҹй…ҚзҪ®дҝЎжҒҜжәҗ
     *
     * @return
     */
    public CorsConfigurationSource corsConfigurationSource() {
        CorsConfiguration corsConfiguration = new CorsConfiguration();
        // и®ҫзҪ®е…Ғи®ёи·Ёеҹҹзҡ„з«ҷзӮ№
        corsConfiguration.addAllowedOrigin("*");
        // и®ҫзҪ®е…Ғи®ёи·Ёеҹҹзҡ„httpж–№жі•
        corsConfiguration.addAllowedMethod("*");
        // и®ҫзҪ®е…Ғи®ёи·Ёеҹҹзҡ„иҜ·жұӮеӨҙ
        corsConfiguration.addAllowedHeader("*");
        // е…Ғи®ёеёҰеҮӯиҜҒ
        corsConfiguration.setAllowCredentials(true);
        // еҜ№жүҖжңүзҡ„urlз”ҹж•Ҳ
        UrlBasedCorsConfigurationSource source = new
                UrlBasedCorsConfigurationSource();
        source.registerCorsConfiguration("/**", corsConfiguration);
        return source;
    }
